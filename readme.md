1. docker build [프로젝트명]
```bash
mvc clean -> mvc package를 하여 target/jar 파일을 생성한 후, build를 실행해줘야 함. 
```
2. docker run -d -p 3000:8080 [프로젝트명]
```bash
여기서 3000:3000으로 했는데, 3000은 도커에 띄울 포트번호이고 8080은 현재 프로젝트의 포트번호임.
기본적으로 스프링부트는 포트번호를 지정하지 않으면 8080의 포트번호를 가져서 접속이 안되었음.
3000:8080으로 하니 잘됨
```
